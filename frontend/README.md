# API Security Gateway Frontend

React dashboard for the API Security Gateway.

## Configuration

Copy `.env.example` to `.env` when the backend is not served from the same origin:

```env
VITE_API_BASE_URL=http://localhost:4000
VITE_GATEWAY_SERVER_URL=http://localhost:4000
```

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```
